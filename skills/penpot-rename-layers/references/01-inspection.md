# 01 ‚Äî Inspection (Phase 0, read-only)

> Loaded when Phase 0 runs. Goal: a complete, classified layer inventory with the signals the inference phase needs. No mutations happen here.

## Order of operations

1. Call `high_level_overview` **once** (loads MCP guidance + a Plugin API overview).
2. Decide **scope** with the user: current page (default), a selected board, or the whole file. Scope determines the `root` you traverse.
3. Run `scripts/inspectLayerStructure.js` via `execute_code` to build the inventory.
4. Cache the result in `storage.run.inventory` and show the user the stats.

Never write in Phase 0. Reading is `penpotUtils.shapeStructure` / `findShapes` only.

## Choosing the traversal root

| Scope | Root |
|-------|------|
| Current page (default) | `penpot.currentPage.root` |
| A selected board | `penpot.selection[0]` (verify it is a board with `shapeStructure`) |
| Whole file | iterate `penpotUtils.getPages()`, traverse each page root |

`penpotUtils.shapeStructure(root, maxDepth)` returns `{ id, name, type, children?, layout? }` ‚Äî the canonical tree read. For a flat list with extra props, use `penpotUtils.findShapes(() => true, root)` and map each shape. Use `penpotUtils.analyzeDescendants(root, evaluator, maxDepth)` when you need each node's **parent context** carried into the evaluator (useful for "is this text near an input?").

## Name-status classification

Every layer lands in exactly one bucket. This is the spine of the whole skill ‚Äî it decides what autofix may touch.

```js
const isAutoGenerated = n =>
  /^(Board|Frame|Rectangle|Group|Ellipse|Text|Path|Bool|Image|Component|Vector)\s*\d*$/i.test(n || "");
const isSemantic = n =>
  /^(header|footer|nav|main|section|article|aside|form|dialog|figure|button|input|select|label|a|h[1-6]|p|small|caption|span|ul|ol|li|img|svg|icon|hr)([-_ ].*)?$/i.test(n || "");
const nameStatus = n => isSemantic(n) ? "semantic" : isAutoGenerated(n) ? "auto-generated" : "custom";
```

- **`auto-generated`** ‚Äî the safe-set. `Rectangle 12`, `Group 4`, `Board`. Autofix may rename these.
- **`semantic`** ‚Äî already a valid element or role name. Leave alone unless it is *wrong* (then it becomes a review row).
- **`custom`** ‚Äî a human-authored non-semantic name (`Hero copy v2`, `final FINAL`). Never auto-overwrite; always review.

## Signals to collect per layer

The inference phase reads these, so collect them now (one `execute_code` call):

| Signal | Source | Used to infer |
|--------|--------|---------------|
| `type` | `shape.type` | text vs. rect vs. path vs. board vs. group |
| `x`/`y`/`width`/`height` | `shape.bounds` (read-only) | position (top‚Üíheader, bottom‚Üífooter), full-width, button-sized |
| `isFullWidth`, `isAtTop`, `isAtBottom` | derived vs. page bounds | structural role |
| child `type`s + count | `shape.children` | "contains nav links", "wraps repeated cards" |
| `characters` (first ~60) | text shapes | label vs. paragraph vs. heading text |
| `fontSize`, `fontWeight` | text shapes | heading level vs. body vs. caption |
| `fills` flags (`hasSolidFill`, `hasImageFill`, `fillColor`) | `shape.fills` | `img` (image fill), `button` (solid fill) |
| `hasStrokeOnly` | `strokes` present, fills empty/transparent | `input` (outline only) |
| `borderRadius` | `shape.borderRadius` | pill/button hint |
| `isComponentMainInstance()` | method | true for the main instance + its internals ‚Üí SKIP; copy instances ‚Üí renamable |
| `layout` / `layout.dir` | board flex/grid | list vs. row of actions |

Read dimensions from `shape.bounds` / `shape.width` / `shape.height` ‚Äî they are read-only, which is fine here. Do **not** attempt to change them.

## Component detection

Verify the API with `penpot_api_info('Board', 'isComponentMainInstance')` before branching. The distinction that matters:

- **Copy instance** (`isComponentCopyInstance()` is true, `isComponentMainInstance()` is false) ‚Äî renaming the instance's own layer is fine; it does not affect the main component.
- **Main component shape** (`isComponentMainInstance()` is true ‚Äî for the main instance **and every shape nested inside it**) ‚Äî renaming it renames the component for *all* instances. This is a shared-asset edit and is **excluded** from autofix. Mark these `mainComponent: true` so Phase 2 skips them. ‚öÝÔ∏è The main instance is *also* an instance, so `isComponentInstance()` is true for it ‚Äî never use `isComponentInstance() === false` to find main shapes.

Do not `detach()` anything in this skill ‚Äî detaching is semi-destructive and out of scope.
